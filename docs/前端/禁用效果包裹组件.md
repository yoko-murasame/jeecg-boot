# 禁用效果包裹组件

禁用效果包裹组件, 用于部分业务需求中表单项的控制.

路径: src/components/yoko/DisableBlock.vue

修改历史:
* 2023-06-09: 新增

使用示例:
```vue
<!--permission为系统的按钮权限配置, 有权限时禁用失效-->
<disable-block :disable="!!model.id" permission="admin:button">
  <a-row>
    <a-col :span="12">
      ...
    </a-col>
    <a-col :span="12">
      ...
    </a-col>
  </a-row>
</disable-block>
```
